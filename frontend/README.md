# 🏡 HomeSpace – Frontend

A modern real estate web application that allows users to explore, search, and visualize properties for buying and renting. Built with a focus on performance, usability, and clean UI.

---

## 🚀 Features

- 🔍 Browse Buy & Rent Listings
- 🗺️ Map-based property visualization (Leaflet)
- ❤️ Save / Favorite listings
- ⚡ Fast and responsive UI
- 📸 Image preview & upload support

---

## 🛠️ Tech Stack

- **Frontend:** React 18 + TypeScript
- **Styling:** Tailwind CSS
- **Maps:** Leaflet + OpenStreetMap
- **API Integration:** REST APIs (Spring Boot backend)

---

src/

## ⚙️ Getting Started

### 1. Clone the repo

```
git clone <your-frontend-repo-url>
cd HomeSpace
```

### 2. Install dependencies

```
npm install
```

### 3. Run the app

```
npm start
```

App runs at:

```
http://localhost:3000
```

---

## 🔗 Backend Integration

This frontend connects to the HomeSpace backend APIs:

- `GET /buyListings`
- `GET /rentListings`

👉 Make sure backend is running before starting frontend.
